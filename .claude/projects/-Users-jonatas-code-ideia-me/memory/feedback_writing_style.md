name: feedback_writing_style
description: Writing authenticity rules — no fabrications, no promotional language
type: feedback
---

Never invent details, anecdotes, or impressions. Only use what the user actually shared or what's directly observable in the content.

**Why:** Jônatas values honest, direct writing. Invented content or promotional framing breaks the authentic voice of the blog.

**How to apply:** When drafting or reviewing posts, use only the real impressions and facts provided. No embellishments, no "amazing", "incredible", "powerful" or other over-promotional adjectives. No fabricated quotes, reactions, or outcomes. If there's not enough raw material to write something, ask for more context rather than filling gaps with invention.
